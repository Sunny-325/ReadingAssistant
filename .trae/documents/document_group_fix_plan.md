# 文档分组功能修复计划

## 问题分析

1. **分组数据关联问题**：
   - 当前所有文档在每个分组中都显示，没有实现真正的分组关联
   - 删除一个分组下的文档会影响所有分组
   - 根本原因是 `getGroupDocuments` 函数直接返回所有文档，没有按分组过滤

2. **界面交互问题**：
   - 分组标签左侧缺少展开/折叠按钮
   - 点击分组时右侧区域没有显示该分组下的所有文件
   - 点击分组下的文档时右侧区域没有单独显示该文档

## 修复方案

### 1. 数据结构修改

- **文档分组关联**：
  - 在文档对象中添加 `group_id` 属性，用于关联到具体分组
  - 修改 `appStore` 中的文档操作逻辑，支持按分组管理文档

### 2. 前端代码修改

- **修改 `DocumentView.vue`**：
  - 实现 `getGroupDocuments` 函数，根据 `group_id` 过滤文档
  - 添加分组展开/折叠按钮，使用 Element Plus 图标
  - 实现点击分组显示该分组下所有文档的功能
  - 实现点击文档显示单个文档的功能

- **修改 `appStore.js`**：
  - 添加文档分组关联逻辑
  - 实现按分组获取文档的方法
  - 实现文档移动到指定分组的方法

### 3. 界面优化

- **分组标签样式**：
  - 在分组标签左侧添加展开/折叠箭头图标
  - 优化分组展开/折叠的视觉效果
  - 确保分组操作按钮（加号和更多选项）位置合理

- **右侧区域显示**：
  - 点击分组时，右侧显示该分组下的所有文档
  - 点击分组下的文档时，右侧显示该文档的详细信息
  - 保持分页功能正常工作

### 4. 功能实现

- **文档分组管理**：
  - 实现文档移动到指定分组的功能
  - 实现按分组删除文档的功能
  - 确保文档只在所属分组中显示

- **用户交互**：
  - 优化分组展开/折叠的交互体验
  - 确保点击分组和文档时的响应速度
  - 添加适当的动画效果

## 风险评估

- **数据迁移**：现有文档没有 `group_id` 属性，需要设置默认值
- **兼容性**：修改后需要确保与现有代码的兼容性
- **性能**：按分组过滤文档可能会影响性能，需要优化

## 实施步骤

1. **修改数据结构**：在文档对象中添加 `group_id` 属性
2. **修改 `appStore.js`**：实现分组相关的方法
3. **修改 `DocumentView.vue`**：实现分组过滤和界面交互
4. **测试**：验证分组功能是否正常工作
5. **优化**：根据测试结果进行调整

## 预期结果

- 文档只在所属分组中显示
- 删除分组下的文档不会影响其他分组
- 分组标签左侧有展开/折叠按钮
- 点击分组时右侧显示该分组下的所有文件
- 点击分组下的文档时右侧显示该文档的详细信息
